# PlayMate Frontend

The frontend is a React application built using **Vite**, **TypeScript**, and **Tailwind CSS**. It uses **shadcn/ui** for UI primitives, **Framer Motion** for animations, and **React Three Fiber** for 3D elements.

## 🚀 Getting Started

Ensure you have a `.env` file created by copying `.env.example` and filling in your credentials:
```bash
cp .env.example .env
```

### Development Scripts

Run the Vite development server:
```bash
npm run dev
```

Build the static asset bundle to `dist/`:
```bash
npm run build
```

Preview the production build locally:
```bash
npm run preview
```

Run test suites:
```bash
npm run test
```

Lint files:
```bash
npm run lint
```
